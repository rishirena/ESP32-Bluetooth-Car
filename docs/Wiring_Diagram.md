# Wiring Connections

## ESP32 to Motor Driver

ESP32 GPIO27 → IN1

ESP32 GPIO26 → IN2

ESP32 GPIO25 → IN3

ESP32 GPIO33 → IN4

ESP32 GND → Driver GND

## Motor Driver to Motors

Motor A → Left Motor

Motor B → Right Motor

## Power Connections

Battery + → Motor Driver VCC

Battery - → Motor Driver GND

Common Ground Required
