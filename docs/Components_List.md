# Components List

## Required Components

1. ESP32 Development Board
2. L298N Motor Driver Module
3. 2 × DC Gear Motors
4. Robot Chassis
5. 2 × Wheels
6. Castor Wheel
7. Battery Pack (7.4V–12V)
8. Jumper Wires
9. Power Switch

## Optional Components

- Ultrasonic Sensor
- MPU6050
- Camera Module
- RGB LEDs
- Buzzer
